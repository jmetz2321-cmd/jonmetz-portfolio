## Jon Metz Portfolio

This repository contains the source for my personal portfolio and AI coaching site.

- **Tech stack**: Static HTML, Tailwind CSS via CDN, and a small amount of vanilla JavaScript for scroll reveals, stat counters, and mobile navigation.
- **Content**: Career overview, business impact case studies, and an AI coaching section with Calendly and Leland integrations.

To view the site locally:

```bash
cd jonmetz-portfolio
python3 -m http.server 8765
```

Then open `http://localhost:8765/` in your browser.
